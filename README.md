프로젝트 목적 
---
    캠핑 용품을 비싼가격을 주고 구매하기 부담스러울때 중고로 거래를 하면서 소비를 줄일 수 있고 사용하지 않는 캠핑용품을 팔 수 있다. 
    객체지향을 이용하여 유지보수에 용이하도록 구현
    효율적이고 불필요한 기술은 최대한 제외
    ++서버구축
